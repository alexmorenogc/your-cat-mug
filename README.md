# Your Cat Mug

E-commerce para tazas personalizadas con fotos de gatos.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy

El deployment a GitHub Pages es automático al hacer push a la rama `main`.
