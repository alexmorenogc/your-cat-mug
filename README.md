# Your Cat Mug
An e-commerce application for personalized mugs featuring adorable cat photos.

## About

Your Cat Mug is a demonstration e-commerce application built to showcase modern web development practices. It allows users to browse a catalog of cat-themed mugs, view product details, add items to a shopping cart, and proceed through a simulated checkout process.

## Features

-   **Product Catalog:** Browse a variety of unique cat-themed mugs.
-   **Product Details:** View detailed information, images, and videos for each mug.
-   **Shopping Cart:** Add desired mugs to your cart and manage quantities.
-   **Checkout Flow:** A simplified modal-based checkout experience.
-   **Responsive Design:** Optimized for various screen sizes.
-   **Smooth Animations:** Enhanced user experience with Framer Motion.
-   **Automated Deployment:** Continuous deployment to GitHub Pages via GitHub Actions.

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Routing:** React Router DOM
- **State Management:** React Context API (for Cart)
- **CI/CD:** GitHub Actions

## Run Locally

**Prerequisites:** [Node.js](https://nodejs.org/) (v20+)

1.  Clone the repository:
    ```bash
    git clone https://github.com/user-attachments/your-cat-mug.git
    cd your-cat-mug
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the application:
    ```bash
    npm run dev
    ```
    The application will typically be available at `http://localhost:3000`.

## Deploy

Deployment to GitHub Pages is automated upon pushing to the `main` branch via a GitHub Action.

For deployment to work correctly, you need to enable GitHub Pages in your repository settings and select "GitHub Actions" as the deployment source.
